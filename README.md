# 🧪 TestSpectra

![Release](https://img.shields.io/github/v/release/TestSpectra/testspectra?style=for-the-badge&color=8A2BE2)

TestSpectra is a powerful, modern, all in one automated testing platform designed for developers and QA engineers who demand precision and efficiency.

---

## 🚀 Key Features

- **Intuitive UI**: Modern design with glassmorphism aesthetics.
- **Cross-Platform**: Available for Windows, macOS, and Linux.
- **Scalable Backend**: High-performance Rust-based server.
- **Docker Ready**: Deploy your testing environment in seconds.
- **Extensible Architecture**: Custom step definitions and flexible test runners.
