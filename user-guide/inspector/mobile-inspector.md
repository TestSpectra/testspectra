# Mobile Inspector

Mobile Inspector di TestSpectra adalah alat bantu untuk memudahkan proses identifikasi elemen (element identification) pada aplikasi mobile (Android/iOS).

## 1. Kegunaan Mobile Inspector

- **Elemen UI Scanning**: Memindai (scan) aplikasi mobile Anda untuk mendapatkan informasi tentang ID, Class, XPath, dan properti lainnya.
- **Copy Selector**: Menyalin ID atau atribut elemen yang dipilih secara instan ke dalam Clipboard Anda.
- **Visual Feedback**: Memberikan sorotan visual pada elemen yang sedang dipilih di layar perangkat.

## 2. Cara Menggunakan Mobile Inspector

1. Pastikan perangkat mobile (nyata atau emulator) sudah terhubung ke TestSpectra.
2. Navigasi ke menu **Mobile Inspector**.
3. Pilih perangkat dan aplikasi yang ingin diinspeksi.
4. Klik pada elemen tertentu di layar untuk melihat detail propertinya di panel detektif.

## 3. Integrasi dengan Page Objects

Informasi yang Anda dapatkan dari Mobile Inspector dapat langsung Anda masukkan sebagai **Selector** di dalam Page Object Manager.

> [!NOTE]
> Fitur Mobile Inspector dirancang untuk meningkatkan kecepatan pembuatan script otomatisasi pengujian mobile secara signifikan.
